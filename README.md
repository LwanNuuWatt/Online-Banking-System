# Online Banking System

This is a Java-based online banking system project with Swing GUI.

## Features
- User login & account management
- Deposit, withdraw, transfer functions
- Loan management
- Admin and customer interfaces

## Technologies
- Java
- Swing GUI
- MySQL database
- Maven (pom.xml)

## How to run
1. Clone the repository
2. Open project in IDE (e.g., Eclipse/IntelliJ)
3. Run `Main.java`
